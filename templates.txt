| ====================================================
| ИНСТРУКЦИЯ ПО РЕДАКТИРОВАНИЮ ШАБЛОНОВ ПИСЕМ
| ====================================================
|
| Этот файл содержит все шаблоны писем для генератора запросов в ПВЗ.
| Все строки, начинающиеся с символа "|", считаются комментариями и игнорируются системой.
|
| СТРУКТУРА ФАЙЛА:
| 
| 1. Каждый шаблон начинается с разделителя вида:
|    ====== КОД_ШАБЛОНА ======
|
| 2. Система нумерации кодов:
|    - 0-0: Приветствие (общее для всех писем)
|    - 0-1: Подпись (общее для всех писем)
|
|    - X-Y-Z-W: где
|      X - Тип ПВЗ (1 = Дропофф, 2 = Конечный)
|      Y - Тип запроса (1 = Прямой маршрут, 2 = Возвратный маршрут, 3 = Запрос видео)
|      Z - Статус доставки (0 = базовый шаблон, 1 = принят, 2 = не принят)
|      W - Причина обращения (номер конкретной причины)
|
| 3. СПЕЦИАЛЬНЫЕ РАЗДЕЛЫ:
|    - templ-subjects: Содержит темы писем
|    - dropdown-X-Y-Z: Содержит опции для выпадающих списков
|
| 4. НАЗВАНИЯ ШАБЛОНОВ:
|    После кода шаблона можно указать название, которое будет отображаться в выпадающем списке:
|    ====== 1-1-1-1 ======
|    ====== Не принят системно ======
|    Содержание шаблона...
|
| 5. ПОДСТАНОВКИ В ШАБЛОНАХ:
|    В шаблонах используются следующие переменные:
|    - {orderNumber} - Номер заказа
|    - {orderNumberNote} - Примечание к номеру заказа
|    - {pvzAddress} - Адрес ПВЗ
|
| 6. ВНУТРЕННИЕ КОММЕНТАРИИ:
|    Внутри шаблона можно оставлять комментарии для коллег, начинающиеся с символа "|".
|    Эти строки не будут включены в сгенерированное письмо.
|    
|    Пример:
|    Здравствуйте!
|    | Этот комментарий не будет включен в письмо
|    Основной текст письма...
|
| ====================================================
